# Assets

Place the following files in this directory before building:

| File                  | Size        | Purpose                          |
|-----------------------|-------------|----------------------------------|
| icon.png              | 1024 × 1024 | App icon (iOS & Android)         |
| splash.png            | 1284 × 2778 | Splash screen image              |
| adaptive-icon.png     | 1024 × 1024 | Android adaptive icon foreground |
| favicon.png           | 48 × 48     | Web favicon (if you ever go web) |

## Quick Placeholder

During development you can generate placeholders with:

```bash
npx expo install expo-asset
# or just drop any PNG in here and rename it
```

Recommended tool for icon generation: https://icon.kitchen
